Mapping Deforestation

This project was an effort to learn how to use multispectral imaging from databases like GEE (Google Earth Engine) to display data visually. The goal for this project was to create a map to visualize deforestation by using GEE's tree cover dataset. In this project, I am displaying where deforestation is hitting most in the world. On the map, you can see that places with brighter red have more destruction of trees in the past 20 years. This project means a lot to me since I have always seen trees being taken down around my neighborhood and wanted to find a way to visualize it on a map and show what was happening.

Key Features

  • Easy to Read Maps: By using GEE it is easy to manuver the map and zoom into specefic areas of interst.

Technologies Used

  • GEE: Uses GEE to determine the tree cover and then finds the percent difference in areas of the world. By using GEE we have accurate sources for the data from their satalites and the data goes back many years.
